Artifact is an initiative for the open source 3d printing community. It is a secure github driven model library where users retain control and recognition for thier contributions. Authors can activate and connect github projects to Artifact to confirm themselves in the library. As a distributed production system, anyone can become a producer and resell open source models while honoring tips to the original artist. With an integrated talk system, distributed production nodes and user gratuities, Artefact aims to be a fully loaded, advanced hub for opensource creation.

As a library, it will have multiple browsing modes, and be a great place to discover new and interesting models. Talk thread will be integrated into Arefact, where users can send tips, comment, start discussions, and more! When producers make and sell a modeller's object, the producer is required to tip the artist. Artefact looks to provide the ultimate level of user control, while allowing anyone to participate. It will be possible to explore, and buy open source objects from local producers through a safe escrow and feedback system. Artefact will be sleek as ever, and provide a next generation platform for physibles. Artifact holds the highest respect for open source contributers which makes that it's highest priority, To provide a better platform for a new generation and economy.
