Artifact

Artifact is an initiative for the open source 3d printing movement. It is a github based library. With integrated social and distributed production system, Artifact will provide a next generation platform for physibles. Topics and projects can be discussed, and users are able to earn reputation and rewards. Artifact holds the highest respect for open source contributers which makes that it's number one priority, so the contributor always retains control and acknowledgement for contributions. Authors can activate thier projects on Artifact to confirm themselves in the library.
